本生成器用于生成带有排曲人、舞曲分布以及详细列表的舞曲列表，包括用于内部讨论的txt文件、用于推文制作的xlsx文件以及用于发布舞曲的PDF文件。

压缩包内包含三部分，css文件夹、wkhtmltopdf文件夹、舞曲列表生成器.exe。

按钮功能介绍：

* 更新列表：基于表格中舞曲信息，更新序号、舞曲时长、分布等信息；
* 检查排曲：检查排曲是否符合排曲规则。
  * 快舞曲时长小于4 min，慢舞曲时长小于4 min 30 s，最好也小于4 min
  * 快慢相间规则
  * 拉手舞架型舞相间规则
  * 每种集体舞最多1首
  * 无重复舞曲
* 生成列表：基于表格中舞曲信息，生成舞曲列表PNG文件
* 保存文件：基于表格中舞曲信息，生成用于播放舞曲文件
* 增加分场：在表格中增加分场（该操作不会更改舞曲文件，需配合`保存文件`使用）
* 删除分场：在表格中删除分场（该操作不会更改舞曲文件，需配合`保存文件`使用）
* 增加舞曲：在表格中增加分场（该操作不会更改舞曲文件，需配合`保存文件`使用）
* 删除舞曲：在表格中删除分场（该操作不会更改舞曲文件，需配合`保存文件`使用）

* 4. 生成列表，如果只是生成舞曲列表，不需导出舞曲文件，点`Yes`

### 虚拟开发环境安装说明
使用`pipenv`构建虚拟开发环境
```
pip install pipenv            # 安装 pipenv
# pipenv --python 3.11          # 指定 python 版本（可直接通过sync命令安装环境）
# 通过requirements.txt安装包（可直接通过sync命令安装环境）
# pipenv install -r requirements-dev.txt 
pipenv sync                   # 根据 Pipfile.lock 安装虚拟环境
pipenv run python .\main.py   # 运行程序
start pyinstaller.bat         # 打包
# 首次打包时，会出现pyinstaller包与pathlib包冲突错误，需要根据提示卸载pathlib包
```
pipenv 使用说明
```
pipenv --where                 # 列出本地工程路径
pipenv --venv                  # 列出虚拟环境路径
pipenv --py                    # 列出虚拟环境的Python可执行文件
pipenv install                 # 创建虚拟环境
pipenv isntall [moduel]        # 安装包
pipenv install [moduel] --dev  # 安装包到开发环境
pipenv uninstall[module]       # 卸载包
pipenv uninstall --all         # 卸载所有包
pipenv graph                   # 查看包依赖
pipenv lock                    # 生成lockfile
pipenv sync                    # 根据 Pipfile.lock 安装虚拟环境
pipenv run python [pyfile]     # 运行py文件
pipenv --rm                    # 删除虚拟环境
# 生成 requirements-dev.txt文件
requirements.txt && pipenv requirements --dev-only > requirements-dev.txt          

```
